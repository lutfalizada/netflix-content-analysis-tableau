# Netflix Content Analysis Dashboard

This project explores Netflix content using Tableau to understand trends in content production.

Key insights analyzed:

• Distribution of Movies vs TV Shows  
• Growth of Netflix content over time  
• Most common genres on the platform  
• Top countries producing Netflix content
